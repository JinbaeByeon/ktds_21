stream
==========

 - stream 예제 코드
 - Dish
   - 항목: 칼로리, 이름
   - Dish() -> 랜덤한 이름과 랜덤한 칼로리의 음식 생성
   - Dish(string name,int calories) -> 이름이 name이고 칼로리가 calories인 음식 생성
  
 - ExampleCode
   - 항목: Dish의 리스트
   - 기능
     - showMenu() -> Dish 리스트에 존재하는 음식들의 이름과 칼로리를 출력
     - example(int i) -> i에 따라 다른 예제 코드들을 실행
 
 - Main : ExampleCode 실행
