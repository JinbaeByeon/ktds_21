hospital
==========

  - Comvertor
    - 기능: 한글의 종성에 따라 뒤에 붙는 글자를 반환해주는 기능

  - Patient
    - 항목: 이름, 증상, 처방받은 약
  
 - Pharmacy
   - 항목: 환자리스트
   - 기능
     - addPatient(String name,String symptom) -> 해당하는 이름과 증상의 환자를 리스트에 추가
     - prescribe() -> 리스트의 모든 환자들의 증상별로 약을 처방 

 - Program
   - main 실행 -> 약국에 환자를 추가하고 약국에서 환자에게 처방을 해줌
